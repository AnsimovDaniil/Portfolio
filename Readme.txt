В папках программа 1, программа 2 лежат исходные файлы программ в соответствии с заданием. Собираются программы отдельно с помощью команды make и компилятора gcc. 
Обе программы поддерживают команду "EXIT" с клавиатуры, после которой штатно выключаются.
Проект имеет следующие недочеты, которые мне не хватило времени исправить:
1. команда make полностью пересобирает проект каждый раз.
2. при отправке через сокет, вначале отправляется длина сообщения, а затем само сообщение. Однако в функии нет проверки на то, сколько байтов было считано и равно ли их количество длине сообщения. Так что, теоритически возможна потеря данных, которая мало вероятна в коротких сообщениях на одной машине, однако имела бы место в реальной программе.
